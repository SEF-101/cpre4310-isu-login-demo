# QR Code Phishing Demo

Project for CPRE 4310 demoing a QR code phishing attack combined with a fake university login. Shows you what an attacker can take from you but doesnt send the data anywhere. "Help" and "Forgot Password" leads to real ISU sites to trick the user.

## How to run

**Locally:** Open `index.html` in a browser

**GitHub Pages:** Visit https://sef-101.github.io/cpre4310-isu-login-demo/

## For the demo
1. Generate a QR code (maybe about signing up for a free campus event) pointing to the above url
2. Have someone scan it and enter fake credentials
3. Show the reveal popup with all captured data
